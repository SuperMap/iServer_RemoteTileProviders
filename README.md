### 一、简介

iServer_RemoteTileProviders提供了SuperMap iServer聚合在线瓦片地图服务的示例。
备注：这些示例也适用于SuperMap iPortal以及SuperMap iExpress。


### 二、扩展开发

示例都是基于[ProviderBase](https://github.com/cxk910204/iServer_RemoteTileProviders/blob/master/src/remotetileproviders/ProviderBase.java) 抽象类实现的。需要指定一组瓦片地图参数，具体参数说明如下：
需要实现List<RemoteTileMapParameter> remoteTileMapParameters()方法，用户需要提供第三方在线地图的地图参数，包括以下几个：

注意：
1.其中分辨率集合和比例尺集合，至少设置一个，但建议都设置。
2.SuperMap默认瓦片轴方向为右下。如果在线地图的瓦片轴方向和supermap的不同的话,那么瓦片行列号可能涉及到转换。详情请参见[部分瓦片地图服务参数介绍](https://github.com/cxk910204/iServer_RemoteTileProviders/blob/master/%E9%83%A8%E5%88%86%E7%AC%AC%E4%B8%89%E6%96%B9%E5%9C%B0%E5%9B%BE%E6%9C%8D%E5%8A%A1%E5%8F%82%E6%95%B0%E4%BB%8B%E7%BB%8D.docx) 和扩展示例。
